## Prerequisites

- `nodejs` version: 9.5.0
- `npm` version: 5.6.0

## Setup project

```
npm install
```
- set `src` folder as resource folder in you **IDE** to have better navigation

## Run application

- `npm run dev` - run webpack dev server on http://localhost:8080/
- `npm run build` - compile SPA
